About Screener.AI

Screener.AI is a powerful tool designed to help recruiters parse, evaluate, and rank resumes against a specific job description.

How It Works:

  1. Provide your Gemini API Key (stored strictly in local memory and not shared). If you don't have a key, the system will run a local matching simulation.
  2. Enter the target Job Description (you can select one of the presets to try it out immediately).
  3. Upload resume files (PDF, DOCX, TXT) or click the Server Samples to load pre-made candidates.
  4. Configure evaluation weights for Skills, Experience, and Education in the weights panel.
  5. Click Screen Resumes to evaluate all resumes at once. Click any row in the dashboard to review a deep analysis.

Technological Stack:

  • Backend: Python, FastAPI, PyPDF, Python-Docx
  • AI: Google Gemini 1.5 Flash (enforcing structured JSON outputs)
  • Frontend: HTML5, CSS3 (Custom Glassmorphism), Vanilla JavaScript
